# 이미지 생성 프롬프트 (zodiac-clue)

> human view: 20260721-image-prompts.html · 2026-07-21 · **추후 GPT 이미지 2.0에 개별 입력용**

에셋 카탈로그(≈108장)의 각 항목을 생성하기 위한 프롬프트. **생성은 별도**, 여기선 프롬프트만. 컨셉 세부는 `20260721-character-concepts.md` 참조.

## 사용법
- 모든 프롬프트 = **[스타일 프리셋] + [대상 묘사] + [타입 접미사]** 로 조합해 입력.
- 캐릭터는 얼굴/SD/전신/말 **4타입** 각각 생성(정합 위해 같은 캐릭터는 시드·묘사 고정).
- 결과는 투명 배경(아이콘·토큰·SD)·단색 배경(전신)으로.

## 사이즈 규칙 (중요)
- GPT 이미지 2.0의 실제 생성 해상도는 **1024×1024(정사각) / 1024×1536(세로) / 1536×1024(가로)** 3종뿐.
- 아이콘·토큰처럼 작은 최종물은 **1024²로 생성 후 다운스케일**(512·256 등)해서 사용.
- 아래 타입별 표기 = `생성(GPT) → 최종(다운스케일)`.

| 용도 | 생성 사이즈(GPT) | 최종 사이즈 |
|---|---|---|
| 얼굴 아이콘 | 1024×1024 | 512×512 |
| SD 캐릭터(선택창) | 1024×1024 | 768×768 |
| 전신 컨셉 | 1024×1536 | 그대로(2:3) |
| 보드 토큰 = **얼굴 아이콘 재사용(정면)** | — | 256×256(다운스케일) |
| 방 바닥(인게임) | 1024×1024 | 룸 종횡비로 크롭/타일 |
| 장소 배경(소개·제출) | 1536×1024 | 그대로(3:2) |
| 도구(장물) 아이콘 | 1024×1024 | 512×512 |
| 로비/결과 배경 | 1536×1024 | 그대로(16:9 근사) |
| UI 프레임 | 1024×1024 | 그대로 |

> 검수 반영: **보드 말은 별도 생성 안 함** — 2D 탑다운·2.5D 정면 빌보드 모두 정면 얼굴을 쓰므로 얼굴 아이콘을 256으로 줄여 토큰에 재사용. 쿼터뷰 토큰·4방 모션·3D 모델은 **S4+ 미래 단계**용이라 지금 프롬프트 대상 아님.

## 스타일 프리셋 (모든 프롬프트 앞에 붙임)
```
조선 사극풍 채색 일러스트, 한국 전통 한복 기반, 동물 특징을 얼굴·귀·소품으로 은은하게 의인화,
먹·주홍·황금 팔레트, 부드러운 붓터치와 은은한 음영, 따뜻한 화선지 질감, 고급 게임 캐릭터 아트,
과하지 않은 디테일, 깔끔한 실루엣.
```

## 타입 접미사 (캐릭터용)
- **얼굴 아이콘**: `상반신 정면 클로즈업, 원형 아이콘 구도, 중앙 정렬, 단순한 따뜻한 배경, 1024x1024 정사각(→512 다운스케일).`
- **SD 캐릭터**: `2~3등신 귀여운 SD(치비), 전신, 정면, 또렷한 표정, 투명 배경, 1024x1024(→768 다운스케일).`
- **전신 컨셉**: `전신 3/4 서 있는 포즈, 살짝 자세한 의상·소품 묘사, 중립 단색 배경, 1024x1536 세로.`
- **보드 토큰**: 별도 생성 안 함 — **얼굴 아이콘을 256px로 다운스케일해 재사용**(2D 탑다운·2.5D 정면 빌보드 모두 정면 얼굴). ~~쿼터뷰 토큰~~은 S4+ 3D 단계용.

## 1. 캐릭터 (12 × 4타입)
> 각 = 프리셋 + 아래 묘사 + 타입 접미사.

| 캐릭터 | 핵심 묘사(프롬프트 본문) |
|---|---|
| 🐭 생쥐 서생 | 왜소한 서생, 회청색 도포에 먹물 얼룩, 큼직한 앞니와 쫑긋한 쥐 귀, 손엔 붓과 주판, 약삭빠르게 굴리는 눈 |
| 🐮 황소 역사 | 거대한 체구의 장사, 무명 저고리에 굵은 팔뚝, 뿔 모양 두건과 코뚜레 장신구, 뚝뚝하나 선한 눈매 |
| 🐯 호랑이 대감 | 장대한 풍채의 대감, 호피 무늬 배자와 금박 갓끈, 위엄 있는 수염과 부리부리한 눈, 손엔 긴 장죽 |
| 🐰 토끼 낭자 | 고운 다홍 치마저고리 낭자, 길고 쫑긋한 토끼 귀 가체, 총기 어린 큰 눈, 손엔 약과 소반과 부채 |
| 🦎 게코 도령 | 늘씬한 도령, 비취색 두루마기, 이마에 용을 흉내 낸 작은 뿔, 길고 특징적인 손발 끝, 자신만만한 눈웃음 |
| 🐍 뱀 무녀 | 검붉은 무복의 무녀, 방울과 부적, 뱀 비늘 문양 목걸이, 가늘고 긴 눈, 손엔 신칼과 방울 |
| 🐴 말 장수 | 볕에 그을린 장돌뱅이, 행상 차림에 봇짐과 말채찍, 갈기 같은 머리, 늘 웃는 입, 짚신 |
| 🐑 양 목동 | 포근한 무명옷에 양털 조끼, 곱슬한 머리와 순한 눈, 목동 지팡이, 움츠린 어깨 |
| 🐵 잔나비 광대 | 오방색 광대옷과 방울 달린 고깔, 붉은 뺨, 잽싼 손발, 장난기 눈, 손엔 소고나 탈 |
| 🐔 닭 훈장 | 꼿꼿한 도포와 정자관, 볏을 닮은 붉은 갓끈, 긴 수염, 회초리와 서책 |
| 🐶 삽살 포교 | 포교 전복과 벙거지, 허리엔 육모방망이와 오랏줄, 삽살개처럼 덥수룩한 눈썹, 곧은 눈빛 |
| 🐷 돼지 객주 | 넉넉한 풍채에 앞치마 두른 저고리, 불그레한 볼과 사람 좋은 웃음, 됫박·엽전 꾸러미, 허리춤 술병 |

## 2. 장소 (배경 10 + 타일)
> **인게임 방 바닥** = `상단뷰 타일러블 텍스처, 이음매 없음, 1024x1024`(룸 종횡비로 크롭/타일 — 실제 룸은 거의 정사각, 예 5×5셀). **소개·제출용 배경** = `와이드 배경, 인물 없음, 1536x1024`.
> ⚠️ 인게임은 와이드 배경을 룸에 넣으면 크롭돼 어긋남 → **타일/정사각**을 룸 바닥에 쓸 것.

| 장소 | 묘사 |
|---|---|
| 정지(부엌) | 한옥 부엌, 아궁이와 가마솥, 장독대, 잔치 음식 준비하는 정경 |
| 대청마루 | 넓은 대청마루, 굵은 기둥과 대들보, 잔칫상이 차려진 정경 |
| 후원 | 한옥 뒷뜰 정원, 작은 연못과 정자, 수목과 괴석 |
| 사랑방 | 선비의 사랑방, 서안·문갑·병풍, 은은한 등잔 |
| 사랑채 | 손님 접객용 사랑채 건물 내부, 정갈한 방 |
| 서재 | 책으로 가득한 서재, 서가·붓·벼루·두루마리 |
| 안방 | 안주인의 안방, 자개장·이불·경대, 따뜻한 분위기 |
| 행랑채 | 하인들의 행랑채, 소박한 여러 방과 툇마루 |
| 별당 | 외딴 별당, 한적하고 은밀한 별채 |
| 잔치상(중앙) | 성대한 생신 잔칫상, 온갖 음식과 선물이 가득한 상 |

## 3. 도구(장물) 6
> 프리셋 + 묘사 + `아이콘: 원형 아이콘 구도, 투명 배경, 1024x1024(→512)` / `토큰: 쿼터뷰 게임 토큰, 투명 배경, 1024x1024(→256)`.

| 장물 | 묘사 |
|---|---|
| 잡채 | 먹음직한 잡채 한 접시, 알록달록 나물과 당면 |
| 잔치 선물 | 비단 보자기로 곱게 싼 선물 꾸러미, 매듭 장식 |
| 금고 | 자물쇠를 채운 묵직한 나무 돈궤(궤짝) |
| 젓가락 | 가지런한 놋 젓가락 한 벌 |
| 술동이 | 술을 담은 옹기 술동이, 뚜껑과 국자 |
| 떡시루 | 김이 오르는 떡시루, 켜켜이 쌓인 떡 |

## 정합성 팁
- 캐릭터별로 **같은 얼굴·팔레트·화풍** 유지: 첫 생성물을 레퍼런스로 재사용하거나 시드 고정.
- 아이콘/토큰/SD는 **투명 배경**, 전신·장소·배경은 채워진 배경.
- 장물·소품은 캐릭터와 화풍 통일(같은 프리셋).
- 생성 후 파일명 규칙 제안: `<카테고리>/<id>-<타입>.png` (예: `char/rat-face.png`, `room/seojae-bg.png`, `loot/japchae-icon.png`).
