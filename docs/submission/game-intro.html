🐯 zodiac-clue — 호랑이 생신 잔치

십이지 손님 중 잔치 음식·선물을 훔친 도둑을 추리하는 웹 멀티플레이 추리 게임. 빈자리는 AI NPC가 채운다.

🎮 지금 플레이 GitHub 소스 문서

목표

봉인된 정답 누가(도둑) · 무엇을(훔친 것) · 어디서(장소)를 추리해 가장 먼저 정확히 고발하면 승리.

예) "게코 도령이 사랑방에서 잡채를 훔쳤다."

조작

  • 이동: 방향키/WASD/ㅈㅁㄴㅇ — 턴 시작 시 주사위(2d6)만큼 복도 이동(방 안·잔치상 무료). 방에 들어간 턴엔 못 나감(다음 턴에 나감).
  • 줌 마우스 휠 · 자유시점 Space(꾹) 드래그.
  • 방에서 [제안] → 도둑·훔친 것 지목(용의자·장물 소환) → 시계방향 반증. [고발]·[통로]·[계략]·[턴 종료].

주요 특징

  • AI NPC 자동 충원: 6인 미만이면 봇이 채워 1인도 플레이. 봇은 규칙기반 추리 + LLM 대사.
  • 비밀 통로: 대각 방 3쌍 연결 — 주사위 없이 순간이동.
  • 고정 NPC(계략): 선택 안 된 십이지 6명이 모서리·중앙 근처 고정. 다가가 [계략] → 상대 카드(정답 아님) 1~2장 엿보기 + 이동 보너스(멀수록 큼). 판당 각 1회.
  • 공통 단서: 솔로(사람1+봇) 시 정답 아닌 카드 2장 공개.
  • 증거 노트: 개인 메모 체크(내 패·엿본 카드 자동 제외). 리매치: 종료 후 [다시 하기].

종료 조건

  • 정답 고발 성공 → 승리(결과 화면 + 다시 하기/메인으로).
  • 오답 고발 → 탈락(관전, 반증만 가능). 최후 1인 생존 시 승리.

실행 방법

  • 웹 즉시 플레이(설치 불필요·무료): zodiac-clue.vercel.app. 방 만들기 → 초대 링크 공유.
  • 6인 미만이면 NPC 자동 충원 — 1인도 시작 가능.
  • 로컬: pnpm install && pnpm dev → localhost:5173

AI 활용 요약

NPC의 행동·판정(결정)은 결정론적 규칙엔진, 대사(표현)만 LLM(Gemini). → 상세는 제출물 ④ AI 기술문서.
