# 제출물 ③ — 게임 소개·설명 (zodiac-clue)

> human view: game-intro.html · NHN NAN 2026 사전과제 제출용 (PDF는 .html 인쇄)

## 제목 / 한줄소개
- **zodiac-clue — 호랑이 생신 잔치**
- 십이지 손님 중 **잔치 음식·선물을 훔친 도둑**을 추리하는 **웹 멀티플레이 추리 게임**. 빈자리는 **AI NPC**가 채워 함께 플레이.

## 목표
- 봉인된 정답(**누가=도둑 · 무엇을=훔친 것 · 어디서=장소**)을 추리해 **가장 먼저 정확히 고발**하면 승리.
- 예: "게코 도령이 사랑방에서 잡채를 훔쳤다."

## 조작
- **이동**: 방향키(또는 WASD/ㅈㅁㄴㅇ), 그리드 한 칸씩. 턴 시작 시 **주사위(2d6)** 만큼 복도 이동, 방 안·잔치상은 무료.
- **줌**: 마우스 휠 / **자유시점**: `Space`(꾹) 드래그.
- **버튼**: 제안 · 고발 · 통로(비밀 통로) · 턴 종료.
- 방에 들어가 **[제안]** → 도둑·훔친 것을 지목하면 그 용의자·장물이 방으로 소환되고, 시계방향으로 **반증**.

## 종료 조건
- **정답 고발 성공 → 승리(게임 종료)**, 결과 화면 + [다시 하기]/[메인으로].
- **오답 고발 → 탈락**(관전 모드, 반증만). 최후 1인 생존 시 승리.

## 실행 방법
- **웹 즉시 플레이**: https://zodiac-clue.vercel.app (설치 불필요, 무료).
- 방 만들기 → 초대 링크(`/room/<코드>`) 공유. **6인 미만이면 NPC 자동 충원**(1인도 시작 가능).
- 로컬: `pnpm install && pnpm dev` → http://localhost:5173
