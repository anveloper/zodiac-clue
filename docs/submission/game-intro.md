# 제출물 ③ — 게임 소개·설명 (zodiac-clue)

> human view: game-intro.html · NHN NAN 2026 사전과제 제출용 (PDF는 .html 인쇄)

## 제목 / 한줄소개
- **zodiac-clue — 호랑이 생신 잔치**
- 십이지 손님 중 **잔치 음식·선물을 훔친 도둑**을 추리하는 **웹 멀티플레이 추리 게임**. 빈자리는 **AI NPC**가 채워 함께 플레이.

## 목표
- 봉인된 정답(**누가=도둑 · 무엇을=훔친 것 · 어디서=장소**)을 추리해 **가장 먼저 정확히 고발**하면 승리.
- 예: "게코 도령이 사랑방에서 잡채를 훔쳤다."

## 조작
- **이동**: 방향키(또는 WASD/ㅈㅁㄴㅇ), 그리드 한 칸씩. 턴 시작 시 **주사위(2d6)** 만큼 복도 이동, 방 안·잔치상은 무료. 방은 **입구(🚪)** 칸으로만 출입. **방에 들어간 턴엔 그 방에서 못 나감**(다음 턴에 나감).
- **줌**: 마우스 휠 / **화면 이동**: `우클릭` 드래그(또는 `Space` 꾹).
- **시점(개발 진화 뷰)**: 좌하단 **[▶ 뷰N]** 버튼으로 단계 순환 — 뷰1(2D) → 뷰2(2.5D) → 뷰3(2.5D+에셋). 항상 뷰1에서 시작.
- **버튼**: 제안 · 고발 · 통로 · **계략** · 턴 종료.
- 방에 들어가 **[제안]** → 도둑·훔친 것을 지목하면 그 용의자·장물이 방으로 소환되고, 시계방향으로 **반증**.

## 주요 특징
- **AI NPC 자동 충원**: 6인 미만이면 봇이 채워 1인도 플레이. 봇은 규칙기반 추리 + LLM 대사.
- **비밀 통로**: 대각 방 3쌍 연결(정지↔별당·후원↔안방·사랑방↔서재) — 주사위 없이 순간이동.
- **고정 NPC(계략)**: 선택 안 된 십이지 6명이 **모서리 2 + 방↔방 사이 벽면 4**에 고정 배치. 다가가 **[계략]** → 상대의 카드(정답 아님) 1~2장을 몰래 엿보기 + 이동 보너스(멀리 있을수록 큼). 판당 각 1회. 계략 NPC가 **귓속말 대사**를 흘리는데 **당사자에게만 전문**이 보이고 타인에겐 `(귓속말)`만 표시.
- **공통 단서**: 솔로(사람1+봇) 시 정답 아닌 카드 2장 공개(추리 보조).
- **증거 노트**: 용의자/훔친 것/장소를 개인 메모로 체크(내 패·엿본 카드 자동 제외).
- **리매치**: 종료 후 같은 방에서 [다시 하기]로 즉시 새 판.

## 종료 조건
- **정답 고발 성공 → 승리(게임 종료)**, 결과 화면 + [다시 하기]/[메인으로].
- **오답 고발 → 탈락**(관전 모드, 반증만). 최후 1인 생존 시 승리.

## 실행 방법
- **웹 즉시 플레이**: https://zodiac-clue.vercel.app (설치 불필요, 무료).
- 방 만들기 → 초대 링크(`/room/<코드>`) 공유. **6인 미만이면 NPC 자동 충원**(1인도 시작 가능).
- 로컬: `pnpm install && pnpm dev` → http://localhost:5173
