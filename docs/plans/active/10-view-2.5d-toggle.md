# plan: 2.5D 시점 토글 (Three.js)

status: active · created: 2026-07-21 · src: 라이브 제안(2.5D 살짝 회전 뷰)

## goal
2D(Phaser) 보드를 유지한 채, 토글 버튼으로 **살짝 내려다보는 2.5D(Three.js)** 뷰로 전환. 서버·HUD·입력 규칙 무변경(렌더러만 교체).

## 접근
- 서버 상태는 정수 그리드 `(x,y)` → 렌더 독립. 2.5D는 `(gx,gy)→(x,0,z)` 매핑만.
- 두 렌더러 공존, **활성 뷰만 입력 처리**(Phaser 입력 비활성+캔버스 숨김 / iso 리스너 부착).
- iso 캔버스는 `#game` 바로 뒤(HUD 앞)에 삽입, HUD가 위에 쌓이도록.

## 2.5D 뷰 사양 (`scenes/iso-view.ts`)
- 카메라 피치 42°, 휠 줌(dist 9~34), 현재 턴 추적 + 지연 팬(150/900ms) + 우측 패널 인셋 보정.
- 방=낮은 박스(깊이감)·잔치상·문(발광 타일+문기둥+"입구")·명패, 토큰/장물/계략=이모지 빌보드 스프라이트.
- 말풍선=DOM 오버레이(타자기), 3D 투영으로 위치.
- 이동 게이팅·WASD/화살표 매핑은 2D와 동일(서버 무변경).

## 마우스 팬(2D·2.5D 공통)
- 우클릭/휠클릭 드래그로 화면 팬(Space 없이). 컨텍스트메뉴 차단.
- 2.5D: 팬 오프셋 유지, 턴 전환 시 리센터. 2D: 우클릭 동안 자유시점 팬 후 복귀.
