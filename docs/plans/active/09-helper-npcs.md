# plan: 고정 NPC(계략) + 보너스 능력

status: active · created: 2026-07-21 · src: 라이브 제안(클루 인트리그 카드 변형)

## goal
선택되지 않은 십이지 6명을 맵에 **고정 NPC**로 배치해 다가가면 보너스(계략)를 제공. 추후 다른 역할로 확장.

## 배치 규칙
- 대상 = ZODIAC(12) − 참여자(6) = **6명**.
- **모서리 1~2명**: 강한 이익(엿보기 2장) + 큰 이동 보너스(빠져나올 껀덕지).
- **나머지는 건물 사이 중앙 근처 후보 목록에서 판마다 랜덤**(엿보기 1장).
- 이동 보너스 = **중앙에서 먼 거리 비례**(`max(2, round(dist/2))`) → 멀수록 크게.
- 후보: `HELPER_CORNERS`(4모서리 중 2), `HELPER_MIDS`(건물 사이 8칸 중 4).

## 보너스: 계략(엿보기)
- 인접(체비셰프 ≤1) + 내 턴 + 미사용 → [계략] 버튼.
- 효과: 상대들이 가진(=정답 아닌) 카드 **1~2장을 나만 몰래 확인**(`peek` private) + **이동 보너스**(거리 비례). **턴은 유지**(돌아올 수 있게).
- 엿본 카드는 클라 증거노트에 자동 '제외' 표시. 판당 각 NPC 1회.
