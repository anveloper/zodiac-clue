# 20260719 — 조디악(십이지) 테마 리스킨

> 컨셉: **"호랑이 생신 잔치 — 도둑은 누구?"** 호랑이 대감의 생신 잔치가 열린 대감집,
> 십이지 손님 6인 중 잔치 선물을 훔친 자를 추리한다. (호랑이=진범 반전은 이후 AI 증인 단계)

---

## feat — 카드 데이터 테마 교체 (`packages/shared`)

- **용의자 6** (동물 손님): 생쥐 서생 / 황소 역사 / 토끼 낭자 / 구렁이 대감 / 잔나비 광대 / 돼지 객주
- **수법(흉기) 6**: 놋촛대 / 은장도 / 다듬잇방망이 / 화승총 / 오랏줄 / 부지깽이
- **장소 9** (대감집): 정지(부엌) / 대청마루 / 후원 / 사랑방 / 사랑채 / 서재 / 안방 / 행랑채 / 별당
- 카드 **ID는 로마자(ascii)로 유지**하고 화면 표기는 `LABELS`(한글)로만 매핑 → 테마 교체 시 라벨만 바꾸면 됨.
- `ROOM_REGIONS`의 방 이름을 새 한옥 장소 ID로 교체(맵 지오메트리는 유지).

## feat — 클라이언트 UI 리스킨 (`apps/client`)

- 타이틀/문구를 사극·잔치 톤으로: "🐯 호랑이 생신 잔치 — 도둑은 누구?", 버튼 "잔치 시작" 등.
- 팔레트를 한옥 톤(먹·주홍·황금)으로 조정.
- **제안/고발 입력을 영문 타이핑 → 드롭다운 선택 모달**로 교체(`openPicker`). 용의자·수법·장소를 라벨로 고르므로 리스킨이 그대로 노출되고 오입력 방지.
