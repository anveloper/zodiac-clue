# 설계: 정통 클루 대비 갭 분석 + 추가 구현 제안

> human view: 20260720-clue-gap-analysis.html · 2026-07-20 라이브 피드백 #4

## 목적
현재 구현을 보드게임 클루(Cluedo) 정통 규칙과 대조해 갭을 정리하고, 추가할 가치가 있는 룰(입구·비밀통로·지름길 등)을 우선순위와 함께 제시.

## 요소 대조 (정통 → 우리)
| 요소 | 정통 클루 | 우리 | 상태 |
|---|---|---|---|
| 용의자 수 | 6명 카드 | 참여자 6명 캐릭터 | ✅ |
| 흉기 | 6종(보드 토큰) | 6종(카드만, 토큰 없음) | 부분 |
| 방 | 9개, 문으로 출입 | 9개 + 입구(door) 1칸 | ✅ |
| 정답 봉투 | 용의자·흉기·방 각 1 | 동일 | ✅ |
| 이동 | 2d6 주사위 | 2d6 + 방 안 무료 | ✅ |
| 방 진입 시 | 그 턴 이동 종료 | 계속 이동 가능 | ❌ |
| 제안 | 방에서 용의자·흉기 지목 → 토큰 소환 | 용의자 토큰 소환(흉기 토큰 없음) | 부분 |
| 반증 | 시계방향 1장 비공개 | 동일 | ✅ |
| 고발 | 봉투 대조, 오답=탈락(반증만) | 동일 | ✅ |
| 비밀 통로 | 대각 방 2쌍 연결(주사위 없이 이동) | 없음 | ❌ |
| 탐정 수첩 | 개인 메모 | 증거노트(로컬) | ✅ |
| 미참여 캐릭터 | 이동 안 함(소환만) | NPC가 실제 플레이 | 차이(의도) |

## 주요 갭
1. **비밀 통로 없음** — 클루의 핵심 묘미. 멀리 있는 방으로 순간 이동하는 전략/역전 요소가 빠짐.
2. **방 진입 정지 없음** — 방을 관통해 지나갈 수 있어 방의 "도착지" 성격이 약함.
3. **흉기 토큰 없음** — 제안 시 흉기가 시각적으로 방에 놓이지 않아 몰입도 약함.
4. **문 봉쇄 개념 없음** — 입구가 1칸이라 정통엔 "막힘"이 있는데 우리는 겹침 허용.

## 추가 구현 제안 (우선순위)
### P1. 비밀 통로(secret passage) ⭐
- 대각 반대편 방 2쌍을 통로로 연결(예: 정지↔별당, 후원↔안방).
- 방 안에서 [통로] 버튼/단축키 → 주사위 없이 상대 방으로 즉시 이동, 그 턴 종료.
- 구현: `ROOM_REGIONS`에 `passage?: string`(연결 방) 추가 + 서버 `usePassage` 메시지 + 클라 버튼. 난이도 중, 효과 큼.

### P2. 방 진입 시 이동 종료
- 입구로 방에 들어서면 `stepsLeft=0`(그 턴 이동 끝). 방=도착지 성격 강화.
- 구현: `handleMove`에서 복도→방 진입 시 `stepsLeft=0`. 난이도 낮음.

### P3. 흉기 토큰(보드 표시)
- 흉기 6종을 방에 배치하고 제안 시 해당 흉기를 그 방으로 이동(용의자 소환과 대칭).
- 구현: 흉기 위치 스키마 + 렌더. 난이도 중.

### P4. 지름길 복도 / 중앙 통로
- 현재 넓은 복도를 미로형·중앙 통로형으로 재설계해 이동 전략성 부여.
- 구현: 벽(non-walkable) 도입 필요 → 난이도 높음(맵 리디자인).

### P5. 문 봉쇄
- 입구 칸에 다른 말이 있으면 진입 불가. 난이도 낮음, 정통성↑.

## 권장 로드맵
P2(방 진입 정지) → P1(비밀 통로) 순으로 하면 적은 비용으로 정통성·재미가 크게 오른다. P3/P4는 여력 있을 때.
