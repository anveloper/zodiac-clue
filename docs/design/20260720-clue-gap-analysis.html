🕵 정통 클루 대비 갭 분석 + 추가 룰 제안

현재 구현을 보드게임 클루(Cluedo) 정통 규칙과 대조하고, 추가할 가치가 있는 룰(입구·비밀통로·지름길)을 우선순위와 함께 제시.

주요 갭

  • 비밀 통로 없음 — 클루의 핵심 묘미(원거리 순간이동 역전 요소)가 빠짐.
  • 방 진입 정지 없음 — 방을 관통 가능해 "도착지" 성격이 약함.
  • 흉기 토큰 없음 — 제안 시 흉기가 방에 시각적으로 놓이지 않음.
  • 문 봉쇄 개념 없음 — 입구 1칸인데 말 겹침을 허용.

추가 구현 제안 (우선순위)

P1비밀 통로 (secret passage) ⭐

대각 반대편 방 2쌍을 통로로 연결(예: 정지↔별당, 후원↔안방). 방 안에서 [통로] → 주사위 없이 상대 방으로 즉시 이동, 그 턴 종료.

구현: ROOM_REGIONS.passage + 서버 usePassage + 클라 버튼. 난이도 중 · 효과 큼.

P2방 진입 시 이동 종료

입구로 방에 들어서면 stepsLeft=0. 방=도착지 성격 강화(정통).

구현: handleMove에서 복도→방 진입 시 stepsLeft=0. 난이도 낮음.

P3흉기 토큰(보드 표시)

흉기 6종을 방에 배치하고 제안 시 해당 흉기를 그 방으로 이동(용의자 소환과 대칭).

구현: 흉기 위치 스키마 + 렌더. 난이도 중.

P4지름길 복도 / 중앙 통로

넓은 복도를 미로형·중앙 통로형으로 재설계해 이동 전략성 부여.

구현: 벽(non-walkable) 도입 → 맵 리디자인. 난이도 높음.

P5문 봉쇄

입구 칸에 다른 말이 있으면 진입 불가. 난이도 낮음, 정통성↑.

권장 로드맵

P2(방 진입 정지) → P1(비밀 통로) 순으로 하면 적은 비용으로 정통성·재미가 크게 오른다. P3/P4는 여력 있을 때.
